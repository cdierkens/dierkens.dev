# dierkens.dev

Source code for https://dierkens.dev

## Commands

### `npm run start`

Start the development server.

### `npm run check`

Run a type check.

### `npm run lint`

Lint the codebase.

### `npm run build`

Build the production files.

### `npm run serve`

Serve the production files (after `npm run build`).

### `npm run test`

Start the test runner.

### `plop <generator>`

To run the following commands install `plop` globally or prefix with `npx`

`npm install plop -g`

or

`npx plop <generator>`

#### `plop list`

List the available generators.

#### `plop component`

Generate a component.
